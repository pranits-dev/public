How to Play

  1. Download the file for your operating system.
  2. Windows: Just run the .exe file.
  3. Linux: Right-click the .AppImage, go to Properties > Permissions, and check "Allow executing file as program". Then run it.
  4. Ensure all players are on the same Wi-Fi network.
  5. One player clicks Host, others use the IP address shown to Join.
